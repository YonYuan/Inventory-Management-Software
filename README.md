# Inventory-Management-Software
An inventory management software, using c# and winForm, the data is saved as TXT.

用C# 实现一个小型连锁超市的库存管理

winForm制作的界面，实现超市、仓库、货架、商品的层级关系，以及相应的增删查改操作

打开：打开数据库

粗存：储存数据库

导出：导出txt到指定文件夹
 
增删查改操作：对应层的增删查改

出入库：按id进行出入库

上一级：返回上一级
 
按下相应按钮会有对应的引导

括号中是快捷键操作
